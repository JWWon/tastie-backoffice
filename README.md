# Backoffice of 'Tastie'

> last update : 2020.04.02

## How to start

1. Ask @Jiwoon Won to give permission of project
2. Clone project

```bash
$ git clone git@gitlab.com:tastie/tastie-client.git
```

3. Set `.env` on root directory
4. Download packages

```bash
$ cd tastie-backoffice
$ yarn install
```

5. Start project

```bash
$ yarn start
```
